# PAW Patrol – Quiz (Cloud Run)

Dieses Repo stellt ein kleines PAW-Patrol-Bilderquiz bereit: Bild ansehen, aus drei Namen wählen, danach Steckbrief anzeigen (aus lokalem Datenset).

## Lokales Starten

```bash
python -m venv .venv
# Windows: .\.venv\Scripts\activate
# macOS/Linux: source .venv/bin/activate
pip install -r requirements.txt
python main.py
